# Assignment 02 — JavaScript Tasks

This repository contains four small JavaScript tasks for "Assignment 02".

- **Task1**: Product inventory operations using `Object.keys`, `Object.values`, and `Object.entries`.
- **Task2**: URL filter builder using `URLSearchParams`.
- **Task3**: Weather fetcher using `async/await` and error handling (uses `wttr.in`).
- **Task4**: Modular student grade system split across `grades.js` and `main.js` (ES module).

How to run

1. Open the project folder in VS Code.
2. For Node-style console runs, use:

```bash
node Task1/task1.js
node Task2/task2.js
node Task3/weather.js
node Task4/main.js
```

3. To run in the browser (recommended for Task4 module):

 - Open `TaskX/index.html` in your browser or use Live Server.
 - Open Developer Tools (F12) and check the Console or the page body for output.

Notes

- Task3 uses `wttr.in` for weather data. The city name is URL-encoded before fetching.
- Task4 is an ES module; serve `index.html` via a simple local server or Live Server to avoid CORS/module errors.
